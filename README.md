# 基于中医食疗知识库与 Coze 智能体的智能食疗问答系统

## 项目简介

本项目围绕中医食疗知识的采集、清洗、存储、可视化分析以及智能问答应用展开，构建了一个简单的中医食疗问答系统。

项目首先通过 Python 爬取中医药网中的内科食疗和妇科病食疗相关数据，提取疾病名称、菜品名称、配方、功效和制法等信息。随后对原始数据进行清洗，去除网页噪声、重复内容和格式异常字段，并将清洗后的数据写入 MySQL 数据库。基于数据库中的数据，项目使用 Python 进行统计分析并绘制图表。最后，将清洗后的 CSV 文件上传至 Coze 平台知识库，创建食疗问答智能体，并通过 Python 图形界面程序调用 API，实现用户注册、登录和食疗问答功能。

该项目实现了“爬虫采集 + 数据清洗 + 数据库存储 + 数据可视化 + 智能问答”的完整流程。

---

## 项目功能

### 1. 数据爬取
- 爬取中医药网食疗板块中的疾病及对应食疗方信息
- 采集内容包括：
  - 疾病名称
  - 疾病链接
  - 菜品名称
  - 配方
  - 功效
  - 制法

### 2. 数据清洗
- 清洗菜品名称中的编号、符号和无效字符
- 清洗配方、功效、制法中的无关网页文本
- 删除如“上一篇”“下一篇”“更多关于……”等网页噪声信息
- 生成清洗后的 CSV 文件

### 3. 数据库存储
- 使用 MySQL 存储清洗后的数据
- 建立疾病表、食疗方表、用户表三张核心数据表
- 实现 Python 自动写入数据库

### 4. 数据可视化
- 基于 MySQL 数据库内容进行统计分析
- 绘制图表展示：
  - 菜品名称出现次数前五名饼图
  - 菜品名称出现次数后五名柱状图
  - 疾病对应食疗方数量统计图

### 5. 智能问答
- 将清洗后的食疗数据整理为 CSV 文件
- 上传至 Coze 知识库
- 创建中医食疗问答智能体
- 使用 Python 调用 Coze API 实现问答功能

### 6. 用户系统
- 简单实现注册与登录功能
- 用户登录后进入问答窗口
- 输入问题后可获得 AI 返回的食疗建议

## 项目技术栈

- **编程语言**：Python
- **爬虫技术**：requests、lxml、XPath、正则表达式
- **数据处理**：pandas
- **数据库**：MySQL、pymysql
- **数据可视化**：matplotlib
- **图形界面**：tkinter
- **AI 平台**：Coze 智能体 / 知识库问答
- **其他工具**：GitHub、CSV 文件处理
